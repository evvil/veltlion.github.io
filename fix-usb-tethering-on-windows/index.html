USB 网络共享

2016年8月14日

　　在 Windows 8 及以上操作系统中使用 Android 手机的 USB 网络共享 会使系统变得非常卡顿。今天无意中在 YouTube 上面看到了解决方法。

  1. 打开设备管理器。
  2. 插上手机，打开 USB网络共享。
  3. 设备管理器–>网络适配器–>Remote NDIS xxxx右键属性。
  4. 选择驱动程序–>更新驱动程序。
  5. 浏览计算机查找–>从计算机驱动程序列表选取。
  6. 取消勾选显示兼容硬件，双击Microsoft–>USB RNDIS6 Adapter。
