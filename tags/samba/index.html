　　正好看到有人做了 Samba4.1.5 的编译脚本和补丁，改改就能拿来用了。
　　如果用 WSL 的话一定要先去掉 $PATH 中 Windows 传进来的环境变量，不然碰到带空格的 Windows 变量会出错。
